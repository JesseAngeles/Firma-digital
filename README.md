# Firma Digital

Aplicación en Java para la firma digital de documentos PDF utilizando la librería iTextPDF.

## Tecnologías

- Java
- iTextPDF 5.4.0
- NetBeans

## Ejecución

Abrir el proyecto en NetBeans y ejecutar la clase principal desde `src/`.

Los documentos de prueba están incluidos en la raíz del proyecto.
